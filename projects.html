Ramen app

This is a final group project for Digital Design class. We used Adobe XD and Illustrator. The concept of this app is to make the user feel comfortable, by having the grandma, the app's character, guide the user in making food. Once you click on the app, the user is notified that they can order the needed ingredients and have it delivered. When the user chooses a food, there is a "step-by-step" mode rather than a video, so that the user and the grandma are both on the same page. This mode also helps when the user is unfamiliar with the making process. The homepage includes various ethnic foods to be chosen from. The "Settings" tab includes account information, such as contact and card info. In "My Profile," one can see their previous and processing orders, and a tracking of their activity.
